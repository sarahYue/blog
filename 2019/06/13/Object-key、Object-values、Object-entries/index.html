Object.key、Object.values、Object.entries

发表于 2019-06-13

例子引发

Object.values

Object.values()方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键值。

Object.keys

Object.values()方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键名。

Object.keys(recipe) // ["apples", "flour", "sugar", "milk", "oil"]

Object.entries

Object.entries方法返回一个数组，成员是参数对象自身的（不含继承的）所有可遍历（enumerable）属性的键值对数组。

Object.entries(recipe) // 自行控制台打印
